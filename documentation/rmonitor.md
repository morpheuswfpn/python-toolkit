# Resource Monitor
A subproject of the bigger toolkit. It uses *psutil* to get information about hardware components which are then displayed using ttk widgets such as labels and progressbars.
## Other Info
It measures:
- RAM
- CPU
- DISK
- BATTERY
In an interval of 0.5 seconds.
# Dependencies
- Tkinter (builtin)
- psutil
- sv-ttk
